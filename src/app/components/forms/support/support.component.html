Support

If you have a problem or need help with the wallet there are several ways of contacting the CasinoCoin team:

Check if you can find your answer at the CasinoCoin FAQ. Join our Discord chat channel via the channel invite. Contact us via social media. You can find all links at the bottom of the CasinoCoin Website. If you encouter a problem, error or bug with the wallet software you can file an issue at the GitHub project website. You can fill out the contact form at the foundation website. Send an email to the support@casinocoin.org email address.
